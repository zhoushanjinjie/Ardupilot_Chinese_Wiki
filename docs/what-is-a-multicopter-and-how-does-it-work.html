多旋翼是什么以及如何工作的？¶

多旋翼飞行器是一种机械简单的飞行器，其运动由多个向下推进的电机/螺旋桨单元加速或减速来控制。

概述¶

多旋翼在空气动力学上是不稳定的，绝对需要一台机载计算机（也称为自动驾驶仪）才能稳定飞行。 因此，它们是“线控飞行”系统，如果计算机不工作，你就不能飞行。 自动驾驶仪结合了来自小型车载MEM陀螺仪和加速度计（与智能手机中的加速度计相同）的数据，以保持对其方向和位置的准确估计。

上面显示的四旋翼机是最简单的多旋翼机类型，每个电机/螺旋桨的旋转方向与两侧的两个电机相反（即，机架对角的电机的旋转方向相同）。

四旋翼机可以通过使一侧的两个电机加速而使另一侧的电机减速来控制其滚转和俯仰旋转。 因此，例如，如果四旋翼机想向左滚动，它会加快机架右侧的电机速度，并减慢左侧的两个电机速度。同样，如果它想向前旋转，它会加快后两个电机的速度，减慢前两个电机。

直升机可以通过加速斜对面的两个电机，并减速其他两个电机来向左或向右转弯（也称为“偏航”）。

水平运动是通过暂时加速/减速一些电机来实现的，这样车辆就朝着所需的行驶方向倾斜，并增加所有电机的总推力，这样飞行器就会向前行驶。一般来说，飞行器倾斜得越多，行驶速度就越快。

高度是通过同时加速或减速所有电机来控制的。

多旋翼和无人机的区别是什么？¶

当多翼飞机能够自主飞行时，它就变成了无人机或无人机。通常情况下，这意味着获取加速度计和陀螺仪信息，并将其与气压计和GPS数据相结合，以便飞行控制器不仅了解其方向，还了解其位置。
