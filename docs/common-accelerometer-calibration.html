加速度计校准¶

本文展示了如何执行基本的加速度计校准（使用Mission Planner）。飞控中的加速度计必须进行校准，以校正其在所有三个轴上的偏置偏移，以及任何离轴变化。

注意

ArduPilot中的加速计校准是必要的。

重要

载具处于解锁状态时，无法执行加速计校准。

校准步骤¶

警告

如果飞控板安装在非标准方向（即箭头未指向前方），则在进行加速计校准之前，请确保AHRS_ORIENTATION设置正确。

提示

对于非常大的载具，这可以在工作台上完成，在设置了如何将其安装在载具中的方向，并在安装后完成校准方向。安装后，请务必使用以下说明中的“校准水平”步骤。

  • 在 初始设置 | 必要硬件，从左侧菜单中选择 加速度计校准

  • 单击 加速度计校准 开始完整的3轴校准。

    Mission Planner将提示您在校准期间将载具放置在每个轴上。按任意键指示飞控已就位，然后继续下一个方向。

    校准位置为: level(水平), right(右侧), left(左侧), nose down(机头向下), nose up(机头向上), 和 back(背面).

    每按一步按键后，车辆必须立即保持静止。这比使角度精确向右更重要，即向左与水平成90度等。除了第一个“LEVEL”，位置可以在精确的20度以内。当你按下按键时，保持每个位置都是非常重要的。

    • 如果可能的话，您应该校准安装在载具上的飞控板。

    • 但是，如果载具的尺寸/形状使安装板变得困难，则可能需要在安装飞控板之前对其进行校准。

    • 水平位置是最重要的，因为这将是你的控制器在飞行时认为水平的姿态。

    • 安装飞控并准备飞行后，您可以使用Mission Planner重新校准此级别位置。将载具置于水平飞行姿态，然后使用Calibrate Level(校准水平) 按钮。

提示

这个Calibrate Level(校准水平)操作只能校正载具中初始校准和最终位置之间的10度差异，并且只能校正俯仰和滚转差异，而不能校正偏航。

  • 到达并保持静止后，使用Click when Done(完成时单击)按钮继续执行所需的位置。

  • 完成校准过程后，Mission Planner将显示“calibration Successful！”，如下所示。

Note

If your autopilot has a built-in IMU heater, then it is recommended that the IMU Temperature Calibration also be done. Some autopilots have this calibration done at the factory in which case, this calibration need not be done.

简易校准¶

Sometimes, for very large vehicles, it’s not easy to do the full 3-axis calibration. In this case, the Simple Accel Cal can be done with the vehicle held still and in a level attitude. This only calibrates the main offsets of the accelerometers, not the minor off-axis variations, so it’s not ideal in terms of optimal performance, but is sometimes an acceptable compromise.

Note

This is NOT the same as the Calibrate Level function. To use that function, either a full 3-axis or simple calibration must be done first!
