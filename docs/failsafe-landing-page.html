Failsafe¶

Copter has a number of failsafe mechanisms to ease vehicle recovery/prevent wandering in the event that vehicle control is lost. The main failsafe topics are listed below.

Note

Once a Radio Failsafe, Battery Failsafe, GCS Failsafe or Terrain Data Loss Failsafe is entered and the failsafe action taken involves a mode change to the vehicle, it remain in that mode until the pilot changes the mode directly.

  • Radio Failsafe
  • Battery Failsafe
  • GCS Failsafe
  • EKF Failsafe
  • Dead Reckoning Failsafe
  • Vibration Failsafe
  • Dead Motor Takeoff Prevention
  • Terrain Data Loss Failsafe (see section on loss of terrain data induced RTL)
  • Crash Check
  • Parachute
  • Independent Watchdog
