Communication Channels¶

It is possible to directly communicate with members of the ArduPilot community, including developers, through the following channels:

Support¶

  • User support requests should be posted to the Discuss Forums (staffed by volunteers)

  • Flight code enhancement requests and known bugs should be added to the ArduPilot Issues list

  • Ground Station enhancement requests and bug reports should be added to the relevant issues list:

    • Mission Planner Issues list

    • APM Planner 2.0 Issues list

    • MAVProxy Issues list

  • Request for professional paid support should be directed to one of the commercial support companies

Development¶

  • Mon/Tue Weekly Meetings on Discord

  • ArduPilot Discord (live chat) live chat for ArduPilot developers (not for support questions)

Facebook¶
