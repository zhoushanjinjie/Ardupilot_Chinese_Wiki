选择地面站¶

本页提供了可用地面控制站（GCS）的高级概述，并提供了链接，以便您做出合适的选择。

概述¶

地面站通常是一种在地面计算机上运行的软件应用程序，通过无线遥测 与无人机通信。它显示无人机性能和位置的实时数据，可以作为“虚拟驾驶舱”，显示许多与驾驶真实飞机时相同的仪器。地面站还可用于控制飞行中的无人机，上传新的任务命令和设置参数。它还经常用于监控无人机摄像头的实时视频流。

至少有十个不同的地面控制站。桌面上有（Mission Planner、APM Planner 2、MAVProxy、QGroundControl和UgCS。 对于平板电脑/智能手机，有Tower（DroidPlanner 3）、MAVPilot、AndroPilot和SidePilot，可用于与ArduPilot（即 Copter, Plane, Rover, AntennaTracker）通信。

  • 准备飞行 的用户可能更喜欢Tower（Droid Planner 3）或其他在平板电脑或手机上运行的GCS的便携性和易用性。

  • DIY/Kit 用户和开发人员通常必须访问配置和分析工具，因此（至少在最初）需要Mission Planner, APM Planner 2或其他功能更全的GCS。

提示

该wiki主要指使用Mission Planner进行设置和配置，因为它是最早创建的，是功能最全的GCS。

功能齐全，应用广泛的GCS。

MAVProxy¶

Plane开发人员经常使用Linux GCS。主要是一个带有图形模块的命令行界面，用于地图和任务编辑。用Python编写，可通过Python模块进行扩展。

  • Platform: Linux

  • Licence: Open source (GPLv3)

  • Wiki

  • 问题列表

  • 源码

QGroundControl¶

QGroundControl与支持MAVLink的飞控合作，包括ArduPilot。它在GCS产品中是独一无二的，因为它在所有桌面和移动平台上运行。

  • Platform: Windows, Mac OS X, Linux, Android and iOS

  • Licence: Open Source (GPLv3)

  • 官网

  • 论坛

UgCS - Universal Ground Control Station¶

具有3D接口的通用且易于使用的地面控制台。支持APM、Pixhawk以及DJI、Mikrokopter等其他制造商的无人机。适用于爱好者和专业用户。

它能够同时与多架无人机进行通信和控制。

UgCS支持多个地图层以及不同的地图提供商。UgCS的一些功能包括-DEM导入、ADS-B转发器和接收器支持、点击即用模式、操纵杆模式、图像地理标记和视频录制。UgCS还配备了一个遥测播放器，可以回放所有航班。

UgCS在所有主要机场周围都内置了禁飞区，并能够创建自定义禁飞区。

支持多节点安装，这意味着可以将多个飞行员与现场的UgCS笔记本电脑连接到中央地面控制服务器。

  • Platform: Windows, Mac OS X, Ubuntu

  • Licence: Proprietary with a free licence available as well (UgCS Open)

  • UgCS Website

  • DIY Drones Group for UgCS Users

  • UgCS Github Repository

移动端对比¶

Tower¶

Tower（又名“DroidPlanner 3”）是一款适用于手机和平板电脑的Android GCS。它是为最终用户和爱好者设计的，包括跟踪、“dronies”（即用无人机拍摄的“自拍”）和3D地图的特殊任务等功能。

  • Platform: Android Phones and Tablets

  • Licence: Open source (GPLv3)

  • 下载 Tower (and 3DR Services) from Google Play.

其它地面站硬件¶

Android¶

对于Android设备，您需要：

  • 基于Android的平板电脑，可以充当USB主机。建议使用谷歌Nexus平板电脑，上表列出了不同GCS的兼容硬件列表。

  • SiK Telemetry Radio System 或类似的（美国915Mhz，欧洲433Mhz），包括双向地面和空中遥测装置

  • USB OTG 线

iOS¶

对于iOS设备，您需要：

  • 基于iOS的设备，如iPad或iPhone。如果运行iPad，建议使用手机版，以获得更好的GPS支持。

  • 无人机的Wifi或蓝牙LE链接

  • 通过Wifi和BLE桥接至 SiK Telemetry Radio System

更多信息请参阅iOS App供应商。

Desktop PCs (Windows/Mac/Linux)¶

对于桌面设备，您需要：

  • SiK Telemetry Radio System 或 类似的 (美国915Mhz，欧洲433Mhz) ，包括双向地面和空中遥测单元。
