无GPS导航¶

这些是允许车辆在没有GPS的情况下估计其位置的可用选项。一旦启用，这允许所有自主和半自主模式，就像GPS可用一样。

  • Intel RealSense T265
  • MarvelMind Beacons
  • ModalAI VOXL
  • Nooploop Beacons
  • Nokov室内光学跟踪
  • 光流
  • OptiTrack运动捕捉系统
  • Pozyx Beacons
  • ROS与谷歌制图（仅限开发者）
  • Vicon 定位系统
  • GPS/无GPS 转换

提示

大多数自动驾驶仪中使用的低成本IMU（加速度计、陀螺仪、指南针）漂移太快，无法在没有外部速度或位置源的情况下进行位置估计。换句话说，低成本的IMU本身不足以估计位置。

提示

除了Vicon之外，需要具有超过1MB闪存的板来运行非GPS导航，因为1MB板仍然支持GPS_ININPUT消息，尽管它们不支持GLOBAL_VISION_POSITION_ESTIMATE，所以它们必须使用GPS_INPUT消息来运行。有关详细信息，请参阅固件限制。
