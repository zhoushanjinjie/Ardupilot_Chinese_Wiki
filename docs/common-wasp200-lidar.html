Attollo Engineering Wasp200 Lidar¶

The Wasp200 has a range of up to 200m and an update rate up to 56hz. More specifications can be found here.

Note

Support for this sensor is available in Plane-3.9 (and higher), Copter-3.6 (and higher) and Rover-3.4 (and higher)

Where to Buy¶

  • The Attollo Engineering online store sells the lidar directly

Connecting to the Autopilot¶

For a serial connection you can use any spare Serial/UART port. The diagram below shows how to connect to SERIAL4.

If the SERIAL4 port on a Pixhawk is being used then the following parameters should be set for the first rangefinder:

  • SERIAL4_PROTOCOL = 9 (Lidar)

  • RNGFND1_TYPE = 18 (Wasp200)

  • RNGFND1_MIN_CM = 200

  • RNGFND1_MAX_CM = 20000. This is the distance in centimeters that the rangefinder can reliably read.

  • RNGFND1_GNDCLEAR = 10 or more accurately the distance in centimetres from the range finder to the ground when the vehicle is landed. This value depends on how you have mounted the rangefinder.

If instead the Telem2 port was used then the serial parameters listed above should instead be:

  • SERIAL2_PROTOCOL = 9

Testing the sensor¶

Distances read by the sensor can be seen in the Mission Planner’s Flight Data screen’s Status tab. Look closely for “sonarrange”.
