机型类别和配置¶

多旋翼¶

FRAME_CLASS 和 FRAME_TYPE 参数应设置为与正在使用的机架相匹配。请参阅 连接电调和电机 页面以获取支持的多旋翼机架列表

如果使用mission planner，请选择初始设置， Mandatory Hardware(必要硬件) | Frame Type(机架类型). 如果您正在使用另一个地面站，应该可以通过参数更新屏幕直接设置 FRAME_CLASS 和 FRAME_TYPE 参数。

提示

对于传统直升机，“Heli(直升机)”应该已经被选中，不应该被更改。对于 Single Copter 和 Coax Copter ，应直接从“完整参数列表”中设置 FRAME_CLASS 参数，直到该问题得到解决。

接下来为您的载具选择车架“Type”（类型）。默认类型为X X。

对于Tricopters、Y6、Traditional Helicopters、Bicopters、SingleCopters和CoaxCopters，忽略机架类型。

电机顺序示意图¶

请参阅 连接电调和电机 页面以获取支持的多旋翼机架列表

传统直升机¶

传统的直升机运行不同版本的固件。有关设置和调试信息，请参阅 传统直升机

HeliQuads是一种混合框架，使用传统直升机固件，但有四个电机。
