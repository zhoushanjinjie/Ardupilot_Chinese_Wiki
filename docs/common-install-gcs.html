安装地面站(GCS)软件¶

ArduPilot可以与许多不同的地面站一起使用。

开发人员通常使用 桌面 GCS，因为这些GCS往往可以更深入地访问车辆设置参数和高级调试功能。如果你只是想坐飞机，你可以选择使用在移动操作系统（iOS、Android）上运行的GCS。

以下提供了一些更受欢迎的地面站的安装链接：

  • Mission Planner (Windows, Linux, Android): Install Mission Planner 该GCS是最兼容的，并密切跟踪ArduPilot中的新功能和更新。建议首次使用和超级用户使用。它在Windows环境中运行得最好。

  • QGroundControl (Windows, Mac OS X, Linux, Android and iOS) 这是基于安卓系统的最受欢迎的GCS，在手机等小格式设备上提供最佳演示。不具备Mission Planner的所有功能，但对某些功能有一个更用户友好的界面。在所有平台上运行良好。

    • 下载和安装 (稳定版)

    • Daily builds

  • APM Planner 2 (Windows, Mac OS X, Linux): Install APM Planner 2 - 提供了基本功能，但目前没有得到很好的维护和更新。
