TeraRanger Tower/ Tower EVO (360 Lidar)¶

Note

The Tower is no longer available and has been replaced by the Tower EVO.

The TeraRanger Tower can be used for Object Avoidance in Loiter and AltHold modes. The sensor has a maximum usable range of about 4.5 meters.

The TeraRanger Tower EVO can be used for Object Avoidance in Loiter and AltHold modes. The sensor has a maximum usable range of about 60 meters.

Mounting the Sensor and Connecting¶

The TeraRanger Tower should be mounted on the top of the vehicle so that sensors scans horizontally and its view is not obstructed by any portion of the vehicle including GPS mast, vehicle legs etc.

Configuration through the Ground Station¶

Example setup below shown for first proximity sensor:

  • SERIAL4_PROTOCOL = “11” (“Lidar360”) if using Serial4.

  • SERIAL4_BAUD = “921” if using Serial4.

  • PRX1_TYPE = “3” to enable the TeraRanger Tower, = “6” for Tower EVO (or 0 to disable).

  • PRX1_ORIENT = “0” Default.

More details on using sensors for object avoidance on Copter can be found here.
